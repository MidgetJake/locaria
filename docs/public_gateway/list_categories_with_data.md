# list_categories_with_data

This method returns an array of all categories that have data within the global_search_view

## Parameters

No parameters supported

## Returns

A categories object containing an array of category names

```json
 {
  "categories": [
    "acl_test",
    "LOCARIA_TEST",
    "LOCARIA_TEST_MOD"
  ],
  "response_code": 200
}
```
