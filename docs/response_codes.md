# Response Codes

#### General Codes

- 200 Success
- 201 Invalid API

### Auth

- 300 Auth error
- 310 Public API error
- 311 Secure API error
- 312 Secure API missing Admins group
- 313 Loader API missing Loader group

### Data

- 400 Data null
- 401 Method is not valid

### Public API

* 500 - SQL Exception

### Internal API

* 600 - SQL Exception
* 601 - Update acl error
* 602 - Delete acl error
* 650 - Duplicate group name
